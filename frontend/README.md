# Frontend - Next.js Admin Panel

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build & Run

```bash
npm run build
npm start
```

## Pages

- `/` - Landing page
- `/admin` - Admin panel for managing blocks
- `/embed/:blockId` - Public embed display

## shadcn/ui Integration

Components use shadcn/ui for consistency. Install additional components as needed.
